Formating Text in Markdown

Headings

Like HTML, Markdown supports 6 heading levels. To create a heading, place hash sign(s) before your heading. The number of hashes placed before the heading determines the header level of the heading (one for header level 1(most impoortant heading), six for header level 6 (least important heading)). 

 # Some Heading(h1)#
 ## Some Heading(h2)##
 ### Some Heading(h3)
 #### Some Heading(h4)###
 ##### Some Heading(h5)
 ###### Some Heading(h6)#
closing hashes are optional, they don't even have to be equal to the number of opening hashes ...

Output

  
Some Heading(h1)

  
Some Heading(h2)

  
Some Heading(h3)

  
Some Heading(h4)

  
Some Heading(h5)

  
Some Heading(h6)
Alternatively,placing any number of Equal Signs (=) or Hyphens (-) on the line below any text, will turn it into a header level 1 heading and header level 2 heading respectively. 
  Some Heading(h1)
  ================
  Some Heading(h2)
  -----------------

Output

Some Heading(h1)


Some Heading(h2)

Paragraphs

To write a Paragraph, type the lines of text without identing and separate the Paragraph with one or more Blank lines. 

This is a Paragraph amet consectetur adipisicing elit. Et dignissimos
sapiente debitis fuga corrupti doloribus repellendus, impedit, amet ipsa 
facilis unde facere itaque expedita distinctio officiis atque quod maiores
consequatur!

This is another Paragraph amet consectetur adipisicing elit. Et dignissimos
sapiente debitis fuga corrupti doloribus repellendus, impedit, amet ipsa 
facilis unde facere itaque expedita distinctio officiis atque quod  
maiores consequatur!

Output

Line Break

To add a Line Break,end a line of text with two or more spaces, then type return.

Italic and Bold Text

Text between two asterisks or underscores will be Italicised... 

  *Italic* text
  _Italic_ text

Output

Text between four asterisks or underscores will be Bold... 

  **Bold** text
  __Bold__ text

Output

Make Bold Ttalic text as follows... 

  ***Italicised Bold*** text
  **_Italicised Bold_** text 
  ___Italicised Bold___ text
  __*Italicised Bold*__ text

Output

Strikethrough

Text between four tilde(~~~~) wll be shown with a Strikethrough ... 

The Earth is ~~Flat~~ Round

Output

The Earth is Flat Round

Lists

To create Unordered List,use either asterisks(*), pluses(+) or hyphens(-) as list item markers. Add a space after the list item markers then the list item name... 

  * Monday
  * Tuesday
  * Wensday

  + Monday
  + Tuesday
  + Wensday

  - Monday
  - Tuesday
  - Wensday

Output

To create an Ordered List, use numbers followed by a period as list item markers.Add a space after a period then the list item . The numbers don't have to be in numerical order, but they should start with one. 

  1. Monday
  2. Tuesday
  3. Wensday
  
  1. Monday
  1. Tuesday
  1. Wensday
  
  1. Monday
  5. Tuesday
  50. Wensday

Output

  1. Monday
  2. Tuesday
  3. Wensday
  1. Monday
  2. Tuesday
  3. Wensday
  1. Monday
  2. Tuesday
  3. Wensday

lists can be Nested as follows ... 

 * Galaxies
    * Andromeda Galaxy
    * Whirlpool Galaxy
    * Magellanic Cloud Galaxy
 * Nebulae
    1. Eagle Nebula
    1. Orion Nebula
    1. Rosette Nebula
    1. Cat's Eye Nebula

Output
