Primitive Data Types

Primitive Data Types are predefined types of data. They are the most basic data types provided by a Programming Language. Python has four Primitive Data Types : Integers , Floats , Boolean, Strings.

Integers are negative or positive whole numbers ... 

# integers
student_age = 16

Floating Point (Floats) are Real Numbers with a Decimal Point.. 

# a float
student_height = 6.24

Boolean variables can have either True or False as values.. 

# boolean
is_raining = True
is_sunny = False
note the values start with capital letters.

Strings are a sequence of Characters.In Python, strings are delimited by either Single quotes('') or Double quotes(""). 

# single quotes
a_string ='spam'
print(a_string)
# double quotes
another_string ="eggs"
print(another_string)
To display a string with quotes, use \ to escape quotes (or any special Characters). Alternatively to display double quotes, use single quotes as delimiters and vice versa.
