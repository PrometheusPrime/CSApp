Operators

Arithematic Operators are used in basic arithematic calculations

Addition or Plus (+) Operator : adds the operands.

a = 5
b = 8
print(a + b) #outputs 13

Subtraction or Minus (-) Operator : subtracts one operand from another.

a = 5
b = 8
print(a - b) #outputs -3

Multiplication (*) Operator : multiplies the operands.

a = 5
b = 8
print(a * b) #outputs 40

Division (/) Operator : divides the operands.Returns a floating point number

a = 8
b = 5
print(a / b) #outputs 1.6

Floor Division(//) Operator : divides the operands.Any remaimder will be dicarded . 

a = 8
b = 5
print(a // b) #outputs 1

Exponetial (**) Operator : is used to raise one number to the power of another (Exponentiation) ... 

a = 8
b = 5
print(a ** b) # 85 outputs 32768
A number can be raised to multiple powers, the powers will be evaluated first 
a = 2
b = 3
c = 4
print(a ** b ** c) # 234 outputs 2417851639229258349412352
# 34 will be evaluated first,
# then 2 will be raised to the result of 34

Modulo or Remainder (%) Operator : returns the remainder of dividing the given numbers ... 

a = 8
b = 5
print(a % b) #outputs 3

Assignment Operators are used to assign values to variables...

Equals (=) assigns a value to a Variable

Plus Equals (+=) adds the number on the right side of the equal sign to the value of the variable on left side of the equal sign and then assigns the result to that variable... 

a = 8
a += 5 # same as a = a + 5
print(a) #outputs 13

Minus Equals (-=) subtracts the number on the right side of the equal sign from the value of the variable on left side of the equal sign and then assigns the result to that variable... 

a = 8
a -= 5 # same as a = a - 5
print(a) #outputs 3

Multiplication Equals (*=) multiplies the number on the right side of the equal sign to the value of the variable on left side of the equal sign and then assigns the result to that variable... 

a = 8
a *= 5 # same as a = a * 5
print(a) #outputs 40

Division Equals (/=) divides the variable on left side of the equal sign by the number on the right side of the equal sign and then assigns the result to that variable... 

a = 8
a /= 5 # same as a = a / 5
print(a) #outputs 1.6

Floor Division Equals (//=)

a = 8
a //= 5 # same as a = a // 5
print(a) #outputs 1

Exponentiation Equals (**=)

a = 8
a **= 5 # same as a = a ** 5
print(a) #outputs 32768

Remainder Equals (%=)

a = 8
a %= 5 # same as a = a % 5
print(a) #outputs 3

Comparison Operators are used to compare given Variables. Comparison Operators return a Boolean value .

Greater Than (>) Operator

a = 8
b = 5
print(a > b) #outputs True

Less Than (<) Operator 

a = 8
b = 5
print(a < b) #outputs False

Greater Than or Equal To (>=) Operator

a = 8
b = 5
print(a >= b) #outputs True

Less Than or Equal To (<=) Operator 

a = 8
b = 5
print(a >= b) #outputs False

Equality (==) Operator : compares whether the Operands are Equal.

a = 8
b = 5
print(a == b) #outputs False

Not Equal (!=) Operator : checks whether the Operands are Not Equal.

a = 8
b = 5
print(a != b) #outputs True

Comparison Operators can be used to compare strings. Strings are compared according to the Alphabetical Order of their component Letters (Z is greater than A) ... 

print("bar" < "bus")
print("bar" <= "bus")
print("bar" > "bus")
print("bar" >= "bus")
print("bar" == "bus")
print("bar" != "bus")

Logical Operators are used to chain comparison Operators, they return a boolean value...

And Operator : is used to combine conditions . Returns True if all the conditions are true.

is_raining = True
is_sunny = False
print(is_raining and is_sunny) #outputs False

Or Operator : is used to combine conditions . Returns True if one of the conditions is true.

is_raining = True
is_sunny = False
print(is_raining or is_sunny) #outputs True

Not Operator : inverts a boolean value . If it was True , it becomes False.. 

is_raining = True
print(not is_raining) #outputs False

Logical Operators can be chained and are evaluated according to Their order of precedence.
