Operating On Strings

The Addition Operator (+) is used to join strings together. Numbers can not be added or combined with strings. 

first_name = "Jane "
last_name = "Doe"
name = first_name + last_name
print(name)

Strings can be multiplied by a number, the result is that string repeated n times . A string can not be multiplied by another string. 

i = "I"
i_times_three = i * 3
print(i_times_three)

The len() function returns the length (number of characters) of a string. 

first_name = "Jane"
print(len(first_name))

The strip() method removes whitespace from the beginning and / the end of a string.

first_name = " Jane "
print("Number of characters before striping whitespace")
print(len(first_name))
first_name = first_name.strip()
print("Number of characters after striping whitespace")
print(len(first_name))

The replace() function returns the length (number of characters) of a string.
