Working With Text

Headings : The H element is used to create headings in HTML. HTML has 6 levels of headings (h1-h6),h1 is the most important(largest) heading and h6 the least important(smallest).

h1 - h6

<!DOCTYPE html>
<html>
 <head>
   <title>Hello HTML</title>
 </head>
 <body>
   <h1>Hello There !!</h1>
   <h2>Hello There !!</h2>
   <h3>Hello There !!</h3>
   <h4>Hello There !!</h4>
   <h5>Hello There !!</h5>
   <h6>Hello There !!</h6>
 </body>
</html>
