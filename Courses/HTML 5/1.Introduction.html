HTML

HTML (HyperText Markup Language) is used to create web pages. HTML is not a Programming Language, it is a Markup Language which uses Tags to discribe how content should be rendered by a program( like a Browser).

Tags usually come in Opening and Closing Tag pairs with the content in the middle.Tag names are surrounded by angle brackets, the Closing Tag has a forward slash before the tag name.

<OpeningTag> Content </ClosingTag>

HTML Element are made up of the Opening Tag and Closing Tag. Empty Elements do not have Closing Tags. Elements are either Block Level Elements (displayed with line breaks) or Inline Elements( displayed without line breaks).

HTML Documents can be created using a simple Text Editor, save the file with either .htm or .html as the file extention. When saving the file, if the Text Editor you're using has an option to set character encoding, set it to UTF-8.

Sample Code

  
<!DOCTYPE html>
<html>
 <head>
  <title>Hello HTML</title>
 </head>
 <body>
  <!-- Some Comment -->
  Hello There !!
 </body>
</html>

Copy and Paste the code above into your Text Editor. In the example the file is named as Hello.html , Name the file whatever you prefer with .html or .htm as the file extention and then save it...

Go to the location where you saved the file and double click it to open it ...

The file will open in your default Browser...

Sample Code Simplified

<!DOCTYPE html> : Browsers can open different types of documents (like pdf et cetra). This declaration tells the browser that the Document type is HTML (and the version of HTML used), so that the document can be rendered correctly.

<html></html> : Marks the beginning and end of the HTML Document, anything outside these tags isn't part of the document.

<head></head> : The head's content is not display on the page, they provides more information about the document which is important to search engines and browsers. Head also contains links to other resources (CSS, Javascript etc). More on this later.

<title></title> : Contains the title of the document which is shown in the browser's title bar or the page's tab, not on the page.

<body></body> : All visible content ( text, images, video, links et cetra) on the web page is contained in the body tags.

<!-- --> : Anything placed between these tags is a comment. Comments are not displayed by the Browser,they are mainly used to document the HTML Document. Note the exclamation point (!) in the opening, but not in the closing tag.
