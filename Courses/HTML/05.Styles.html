Styles

Styles are used to control the way the HTML Document is presented(text color, background color,font type, et cetra).

Inline Styles

The Style Attribute is used to change the style of an HTML element. It has the following syntax : 

style = "property:value;"
the property is what you want to change on an element. 

<!DOCTYPE html>
<html>
  <head>
    <title>Hello HTML</title>
  </head>
  <body>
    <p style = "color:aquamarine;">
    This is a paragraph.
    </p>
    <p style = "color:teal;">
    This is another paragraph.
    </p>
  </body>
</html>

You can add multiple styles on an element.

<!DOCTYPE html>
<html>
  <head>
    <title>Hello HTML</title>
  </head>
  <body style="background-color:#7f7f7f;">
    <h1 style="text-align:center;color:rgb(255,255,255);">Hello There !!</h1>
    <p style="color:aqua;font-family:'Times New Roman';font-size:40px;">
    This is a paragraph.
    </p>
    <p style="color:aquamarine;font-family:verdana;font-size:30px;">
    This is another paragraph.
    </p>
  </body>
</html>

Adding styles to an element with the style attribute is called Inline Styling.

Internal/Embeded Styles

You can also style an HTML Document with the <style> element. The style element is nested inside the head element. Styling an HTML Document with the style element is called Internal Styling. It follows the following syntax :

<style>
selector {property:value;}
</style>
The selector is the HTML element you want to style , property:value; is called Declaration. 

<!DOCTYPE html>
<html>
 <head>
    <title>Hello HTML</title>
    <style>
    body{background-color: darkgrey;}
    h1{text-align:center;font-family:'Times New Roman';color:darkcyan;}
    p{color:aquamarine;font-family:verdana;}
    </style>
 </head>
 <body>
  <h1>
  This is a Heading.
  </h1>
  <p>
  This is a paragraph.
  </p>
  <p>
  This is another paragraph.
  </p>
 </body>
</html>

You can add same styles to multiple elements by comma seperating each selector.

<!DOCTYPE html>
<html>
 <head>
 <title>Hello HTML</title>
 <style>
 h1,h2{
   text-align:center;
   font-family:'Times New Roman';
   color:darkcyan;}
 </style>
 </head>
 <body>
 <h1>
 This is a Heading.
 </h1>
 <p>
 This is a paragraph.
 </p>
 <h2>
 This is another Heading.
 </h2>
</body>
</html>

Class and Id Selectors

The Class Selector is used to style a specific element(s).A class can be used as many times as needed,on any number of elements. Use the class attribute to add a class to an element.To select an element(s) with a class, use the period (.) character, then follow it with the class name. A class name cannot start with a number.

<element class="className"></element>

An element can belong to multiple classes, seperate the classes with a space.

<element class="classA classB"></element>

The Id Selector is used to style a specific element.An id is used only once per page ,it is used to select one unique element. Use the id attribute to add an id to an element. To select an element with an id, use the hash (#) character, then follow it with the id. An id name cannot start with a number.

<element id="idName"></element>

Class and Id Selectors Example

<!DOCTYPE html>
<html>
  <head>
   <title>Hello HTML</title>
   <style>
    .myClass{
     text-align:center;
     font-family:'Times New Roman';
     color:darkcyan;}
     #myId{
      color:darkslategray;font-family:verdana;}
   </style>
 </head>
 <body>
   <h1 class="myClass">
   This is a Heading with Class.
   </h1>
   <p id="myId">
   This is a paragraph with an Id.
   </p>
   <p class="myClass">
   This is paragraph with Class.
   </p>
   <p>
   This is another paragraph.
   </p>
 </body>
</html>

Inheritance

A child element will inherit styles from its parent(an element in which the child element is nested in).Applying styles to a child element will overwrite the inherited ones,the most specific style rule will be applied to an element.

<!DOCTYPE html>
<html>
  <head>
    <title>Hello HTML</title>
    <style>
      body{
       color:#7f7f7f;}
     h1{
       text-align:center;
       font-family:'Times New Roman';
       color:darkcyan;}
      #myId{darkseagreen
       color:darkseagreen;font-family:verdana;}
    </style>
  </head>
  <body>
    <h1>
    This is a Heading with specific styles.
    </h1>
    <p>
    This paragraph is inheriting from parent.
    </p>
    <h2>
    This Heading is inheriting from parent.
    </h2>
    <p id="myId">
    This is a paragraph with specific styles.
    </p>
  </body>
</html>
